今天结束时，至少要达到下面结果：

## 3. 今天的具体执行顺序

建议严格按这个顺序推进：

1. 先清理 battle 默认卡组与调试入口
- 明确今天要进 battle 测哪些牌
- 保证调试牌组顺序固定，方便反复验证

2. 补卡牌第一批
- `card_stance`
- `card_heavy_blow`
- `card_quick_shot`
- `card_concussion_shot`
- `card_weathering`

3. 做普通攻击 / 武器 / 天赋影响联动复核
- 普通攻击受武器影响
- 防御动作受减伤影响
- 移动力受装备 / 天赋影响
- `拔枪` 临时武器链路不被新改动破坏

4. 装备第一批落地
- `equip_old_coat`
- `equip_magnetic_scabbard`
- `equip_phase_boots`
- 视时间补 `equip_red_scarf`、`equip_target_lens`

5. 敌人第一批差异化
- `pirate_blocker`
- `pirate_scout`
- `pirate_shocker`
- 有余力再补 `pirate_gunner`

6. 最后做 debug 场景回归
- 固定房间
- 固定牌组
- 固定装备切换测试
