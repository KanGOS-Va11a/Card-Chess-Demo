# 流程节点配置表（首版）

日期：2026-03-28

## 字段说明
- node_id：节点唯一标识
- phase：所属阶段
- type：tutorial | hub | encounter | event | gate | boss_prelude | wrapup
- dependencies：前置节点
- once：是否一次性
- success_next：成功后去向
- fail_next：失败后去向（可空）
- state_key：状态标记键
